# 🌍 Global Sales Data Analysis - Power BI Dashboard

## 📌 Project Overview
This project focuses on analyzing global sales data using Power BI to uncover meaningful insights and support business decision-making. The dashboard provides a comprehensive view of sales performance across regions, products, and time.

---

## 🎯 Objectives
- Analyze sales performance across different countries and regions  
- Identify top-performing products and categories  
- Track sales trends over time (monthly & yearly)  
- Evaluate profit distribution and high-revenue segments  

---

## 📊 Dashboard Features
- 🌍 **Region-wise Sales Analysis** (Map Visualization)  
- 📈 **Sales Trends Over Time** (Line Chart)  
- 🏆 **Top Products & Categories**  
- 💰 **Profit Analysis**  
- 🎛 **Interactive Filters** (Region, Category, Year)  

---

## 🛠 Tools & Technologies Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Visualization  

---



## 📈 Key Insights
- Certain regions contribute significantly to overall revenue  
- Seasonal trends observed in monthly sales data  
- A few product categories dominate total sales  
- Profit margins vary across regions and product types
